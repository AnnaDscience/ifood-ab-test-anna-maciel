# iFood Data Analysis Case

## Objetivo

Este projeto apresenta a análise de um experimento A/B relacionado a uma
campanha de cupons na plataforma do iFood.

O objetivo do estudo é avaliar o impacto da campanha, analisar sua
viabilidade financeira e propor estratégias de segmentação de usuários
para otimizar campanhas promocionais.

A análise foi desenvolvida utilizando Python em ambiente Databricks,
com foco em experimentação, análise de dados e geração de insights
para tomada de decisão de negócio.
